## The Moon
### To actually route to the moon type "scarletdevilmansion" in the terminal
Adds the Scarlet Devil Mansion moon, as an alternative to the snow moons. Intended to be used with the [Scarlet Devil Mansion Interior](https://thunderstore.io/c/lethal-company/p/Alice/ScarletDevilMansion/) with the More loot, More Danger preset. To use allow this interior to spawn on this moon please change the Lethal level loader config. (More information on changing configs can be found on the modpage for the interior).

The moon is a larger moon compared to the vanilla ones, it's price is lower to encourage the purchase of a company cruiser to reduce the traveling times. It is also recommended to use larger lobby sizes.
